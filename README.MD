# The Broward Realtor System

## Overview
Autonomous lead generation platform for Broward County real estate listings with AI chatbot, property valuation tool, and signal-based prospecting.

## Features
- 🤖 AI Chatbot with $400 brokerage advantage messaging
- 🏠 Free Property Valuation Tool
- 📧 Email Nurture Automation
- 🎯 Signal Detection Engine (prospecting)
- 🚀 Cloudflare Pages + Workers Deployment

## Setup
1. Follow deployment script: `./deploy-all.sh`
2. Configure environment variables in Cloudflare Dashboard
3. Point DNS to Cloudflare Pages URL

## Tech Stack
- Frontend: HTML/CSS/JavaScript (React-ready)
- Backend: Node.js TypeScript (Cloudflare Workers)
- Database: Neon PostgreSQL
- AI: Anthropic Claude API
- Email: Resend.com
